Query Parameters


type


The query type

Parameters:

  • search: search for a gene
  • isoform: search for a specific isoform
  • targetEfficiency: retrieve efficiency information for a specific target
  • oligos: retrieve potential oligos for specific target
  • primers: retrieve primers for specific target

Example: /api?type=search



?type=search


Search for a gene by name or id

Sub Parameters:

  • gene: the gene name

Example: /api?type=search&gene=pkn


Return:

  • name: name of searched gene
  • isoForms: a list of isoforms related to this gene
  • url: the flybase url where this information can be access directly
  • sequence: the extended gene sequence returned from flybase
