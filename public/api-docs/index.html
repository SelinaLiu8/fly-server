Get started

    API Endpoint

        https://crisprbuilder.org/api/

Documentation for the CrisprBuilder API

To use this API, you need an API key. Please contact us at info@crisprbuilder.com to get an API key.

GENE


# Here is a curl example
curl \
-X POST https://crisprbuilder.com/api/query \
-F 'key=your_api_key' \
-F 'type=gene,oligos,primers,primer_efficiency' \
-F 'gene=pkn,polo' \

To interact with our API, you need to make a GET request to the following url :
https://crisprbuilder.com/api/query 



Result example :

{
  query: https://crisprbuilder.com/api?type=search&gene=polo
  result: {
    "id":12,
    "name":"polo",
    "isoForms":"[\"polo-RA\",\"polo-RB\"]",
    "url":"https://flybase.org/download/sequence/FBgn0003124/gene_extended",
    "sequence":"AAACAAAAGAGGCCACACGGCATTAGCGA...",
    "isoForm":"polo-RA",
    "time":"2021-09-06 13:42:23",
    "isoFormSequence":"ATGGCCGCGAAGCCCGAGGATAAG..."
  }
}

QUERY PARAMETERS

Field Type Description
type String The type of api call you are making
gene String A search word to find the gene information.
isoForm String (Optional) A search word to find information on a specific isoform by name.

RETURN VALUES

Field Type Description
name String The search term you entered.
isoForms String A list of isoforms associated with the gene.
url String The flybase url from where this info was pulled.
sequence String The extended gene sequence pulled from Flybase.
isoForm String The isoform returned from the query (will return default isoform from flybase is one is not queried).
isoFormSequence String The full sequence of the isoform returned from the query.

Errors

The Crispr Builder API uses the following error codes:

Error Code Meaning
000 Some parameters are missing. This error appears when you don't pass every mandatory parameters.
